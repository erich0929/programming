Design update

We have spent two years introducing a new design for the Guardian website, taking care to ensure improved accessibility. This design has now been applied to the majority of pages, however some still remain such as User registration / sign-in and these will not yet have the same level of accessibility or some of the features listed below.

Text resizing

The modern browsers we support (Firefox 3.6 and later, Internet Explorer 7 and later, Chrome (latest version) and Safari 4.x and later) feature in-built accessibility tools, including zoom and text-resizing.

You can change the text size or adjust the page zoom in your browser by following these instructions:

• PC / Internet Explorer 7 From the menu bar underneath the search field, select Page and point to Text size

• PC / other browsers Increase page zoom: Hold down the CTRL key and press + Decrease page zoom: Hold down the CTRL key and press -

• Mac / all browsers Increase page zoom: Hold down the Command key and press + Decrease page zoom: Hold down the Command key and press -

Access keys

These can be accessed in the following way:

• PC: hold down Alt
• Mac: hold down Ctrl
• Firefox 2: hold down Alt and Shift

Then press one of the keys listed below:

's' to jump to the content
'0' to jump to the site navigation
'4' to jump to the site search
'8' to see our terms and conditions
'c' to jump to the comments (where comments are available).

User commenting

We have recently increased the accessibility of our commenting section by no longer using Javascript to view the comments. A more technical explanation can be found on the Inside the Guardian website blog. Some areas still require Javascript such as reporting abuse and recommending comments.

The following items should serve to further help users in this area:

• Access key 'c' (as mentioned in the section above) has been added to provide a link that moves the user directly to the start of the comments.

• RSS feeds of comments are available so you can view them using an RSS reader, such as Google Reader.

• We have ensured that the text resizing code and other features work with the comments.

• Comments are keyboard navigable.

Recent site improvements

1. Crumb navigation not dependent upon Javascript to display the arrow images.

2. Sliding caption column: changed heading colour to a monochrome scheme to improve readability.

3. Breaking news ticker: lowered its CPU usage and removed the iframe, making it easier to access for users browsing with a keyboard or any assistive technology.

4. Display of comments not dependent upon Javascript.

5. Large text-size wide-layout rolled out across all pages.

National Talking Newspapers

National Talking Newspapers offers Guardian content in a range of audio and full-text formats. Subscriptions are flexible and subscribers can change their choices during their subscription period. CDs and tapes and delivered freepost if the recipient is visually impaired.

More information is available on their website. If you have further queries, please contact National Talking Newspapers' customer services on 01435 866102 or email info@tnauk.org.uk.
